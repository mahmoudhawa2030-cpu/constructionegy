# Construction-Egy Portable Modules

This directory contains reusable modules extracted from the construction-egy app that can be installed in other Next.js applications.

## Available Modules

### 1. Things Counter (`counter/`)
- Counts objects in images using Roboflow API
- Features camera capture, image upload, and crop selection
- Currently configured for pipe counting

### 2. Document Scanner (`scanner/`)
- Full document scanning with edge detection
- Multiple filter options including AI enhancement
- Multi-page support with export functionality

### 3. DWG Viewer (`dwg-viewer/`)
- DWG file viewer with PDF conversion
- Drag & drop file upload
- Uses LibreCAD for DWG to PDF conversion
- PDF preview and download functionality
